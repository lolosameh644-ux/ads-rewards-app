## خطوات النشر

### للـ Android

1. **بناء APK**
   ```bash
   eas build --platform android --type apk
   ```

2. **تحميل على Google Play Store**
   - اذهب إلى [Google Play Console](https://play.google.com/console)
   - أنشئ تطبيق جديد
   - رفع APK
   - ملء المعلومات والصور
   - اطلب المراجعة

### للـ iOS

1. **بناء IPA**
   ```bash
   eas build --platform ios --type archive
   ```

2. **تحميل على App Store**
   - اذهب إلى [App Store Connect](https://appstoreconnect.apple.com)
   - أنشئ تطبيق جديد
   - رفع IPA
   - ملء المعلومات والصور
   - اطلب المراجعة

### للـ Web

1. **بناء الإصدار**
   ```bash
   pnpm build
   ```

2. **النشر**
   - استخدم Vercel أو Netlify
   - ربط مستودع GitHub
   - تفعيل النشر التلقائي

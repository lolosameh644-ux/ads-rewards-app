# تصميم تطبيق Ads Rewards

## نظرة عامة
تطبيق موبايل لمشاهدة الإعلانات وكسب النقاط وسحب الأرباح. التطبيق يستخدم Unity Ads لعرض الإعلانات المكافئة، ويتيح للمستخدمين سحب أرباحهم عبر Instapay أو Vodafone Cash أو PayPal.

## الشاشات الرئيسية

### 1. شاشة تسجيل الدخول (Login Screen)
- **المحتوى**: 
  - شعار التطبيق في الأعلى
  - حقل إدخال البريد الإلكتروني
  - حقل إدخال كلمة المرور
  - زر تسجيل الدخول
  - رابط "إنشاء حساب جديد"
- **الوظيفة**: تسجيل دخول المستخدم باستخدام البريد الإلكتروني وكلمة المرور
- **التدفق**: بعد تسجيل الدخول الناجح → شاشة Home

### 2. شاشة التسجيل (Register Screen)
- **المحتوى**:
  - حقل الاسم
  - حقل البريد الإلكتروني
  - حقل كلمة المرور
  - حقل تأكيد كلمة المرور
  - زر "إنشاء حساب"
- **الوظيفة**: إنشاء حساب جديد
- **التدفق**: بعد التسجيل الناجح → شاشة Home

### 3. شاشة الرئيسية (Home Screen)
- **المحتوى**:
  - بطاقة عرض النقاط الحالية في الأعلى (عدد النقاط + قيمتها بالدولار)
  - زر كبير في المنتصف "شاهد إعلان واربح نقطة" مع أيقونة
  - عداد عدد الإعلانات المشاهدة اليوم
  - معلومات: "كل 300 نقطة = 1 دولار"
  - معلومات: "الحد الأدنى للسحب 900 نقطة = 3 دولار"
- **الوظيفة**: 
  - عرض رصيد النقاط الحالي
  - زر لمشاهدة إعلان Unity Ads
  - عند الضغط على الزر → تشغيل إعلان Unity Ads
  - عند إكمال الإعلان → إضافة نقطة واحدة للرصيد
- **التدفق**: الضغط على الزر → عرض إعلان Unity → إضافة نقطة

### 4. شاشة السحب (Withdraw Screen)
- **المحتوى**:
  - عرض الرصيد الحالي
  - اختيار وسيلة السحب (Instapay / Vodafone Cash / PayPal)
  - نموذج طلب السحب:
    - عدد النقاط المراد سحبها (يجب أن يكون >= 900)
    - رقم الهاتف (لـ Instapay و Vodafone Cash)
    - أو البريد الإلكتروني (لـ PayPal)
  - زر "إرسال طلب السحب"
  - قائمة بطلبات السحب السابقة (الحالة: قيد المراجعة / مكتمل / مرفوض)
- **الوظيفة**: 
  - السماح للمستخدم بطلب سحب النقاط
  - التحقق من الحد الأدنى (900 نقطة)
  - إرسال الطلب إلى قاعدة البيانات
  - عرض حالة الطلبات السابقة
- **التدفق**: اختيار الوسيلة → إدخال البيانات → إرسال الطلب → حفظ في قاعدة البيانات

### 5. شاشة الملف الشخصي (Profile Screen)
- **المحتوى**:
  - معلومات المستخدم (الاسم، البريد الإلكتروني)
  - إحصائيات (إجمالي النقاط المكتسبة، عدد الإعلانات المشاهدة، إجمالي المسحوبات)
  - زر تسجيل الخروج
- **الوظيفة**: عرض معلومات المستخدم وإحصائياته

### 6. لوحة الأدمن (Admin Dashboard)
- **المحتوى**:
  - قائمة بجميع المستخدمين مع:
    - الاسم والبريد الإلكتروني
    - عدد النقاط الحالية
    - عدد الإعلانات المشاهدة
    - زر لتعديل النقاط (إضافة/خصم)
  - قائمة بطلبات السحب:
    - معلومات المستخدم
    - عدد النقاط
    - وسيلة السحب والرقم/البريد
    - التاريخ
    - الحالة (قيد المراجعة / مكتمل / مرفوض)
    - أزرار (قبول / رفض)
  - إحصائيات عامة:
    - عدد المستخدمين
    - إجمالي النقاط الموزعة
    - إجمالي المسحوبات
- **الوظيفة**: 
  - إدارة المستخدمين والنقاط
  - مراجعة طلبات السحب والموافقة عليها أو رفضها
  - عرض إحصائيات التطبيق
- **الوصول**: فقط للمستخدمين ذوي صلاحية Admin

## التدفقات الرئيسية

### تدفق مشاهدة الإعلان
1. المستخدم يفتح شاشة Home
2. يضغط على زر "شاهد إعلان واربح نقطة"
3. يتم تحميل إعلان Unity Ads (Rewarded Ad)
4. المستخدم يشاهد الإعلان حتى النهاية
5. يتم إضافة نقطة واحدة إلى رصيد المستخدم
6. يتم تحديث عداد الإعلانات المشاهدة
7. يظهر إشعار بنجاح العملية

### تدفق طلب السحب
1. المستخدم يفتح شاشة Withdraw
2. يختار وسيلة السحب (Instapay / Vodafone Cash / PayPal)
3. يدخل عدد النقاط (يجب أن يكون >= 900)
4. يدخل رقم الهاتف أو البريد الإلكتروني
5. يضغط على "إرسال طلب السحب"
6. يتم التحقق من الرصيد الكافي
7. يتم خصم النقاط من الرصيد
8. يتم حفظ الطلب في قاعدة البيانات بحالة "قيد المراجعة"
9. يظهر إشعار بنجاح إرسال الطلب

### تدفق مراجعة الطلبات (Admin)
1. الأدمن يفتح لوحة التحكم
2. يرى قائمة بطلبات السحب "قيد المراجعة"
3. يراجع تفاصيل الطلب
4. يضغط على "قبول" أو "رفض"
5. إذا تم القبول: تتغير الحالة إلى "مكتمل"
6. إذا تم الرفض: تتغير الحالة إلى "مرفوض" ويتم إرجاع النقاط للمستخدم

### نظام الألوان
- **اللون الأساسي (Primary)**: أخضر (#10B981) - يرمز للمال والأرباح
- **الخلفية (Background)**: أبيض (#FFFFFF) في الوضع النهاري، رمادي داكن (#151718) في الوضع الليلي
- **السطح (Surface)**: رمادي فاتح (#F5F5F5) في النهاري، رمادي داكن (#1E2022) في الليلي
- **النص الأساسي (Foreground)**: أسود (#11181C) في النهاري، أبيض (#ECEDEE) في الليلي
- **النص الثانوي (Muted)**: رمادي (#687076) في النهاري، رمادي فاتح (#9BA1A6) في الليلي
- **النجاح (Success)**: أخضر (#22C55E)
- **التحذير (Warning)**: برتقالي (#F59E0B)
- **الخطأ (Error)**: أحمر (#EF4444)

### مبادئ التصميم
- تصميم بسيط ونظيف يركز على الوظيفة
- استخدام بطاقات (Cards) لعرض المعلومات المهمة
- أزرار كبيرة وواضحة لسهولة الاستخدام
- استخدام الأيقونات لتوضيح الوظائف
- دعم الوضع الليلي والنهاري
- تصميم متجاوب مع أحجام الشاشات المختلفة
